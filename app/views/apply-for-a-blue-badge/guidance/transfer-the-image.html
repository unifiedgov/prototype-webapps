Transfer the photo

{% set yesHTML %}

If you took the photo on this device, you can continue and upload directly from this device.

When you return to {{your}} {{application}} there will be a link that will allow you to "Select a file from this device". Use this to locate the photo on this device.

{% endset -%} {% set noHTML %}

If you haven't already saved the photo to this device, you'll have to transfer it, by either:

  • sending it to yourself by email
  • attaching a cable between the camera and this device
  • using Bluetooth to move the photo from your mobile phone or tablet to this device

You can continue once you have saved the photo as a file to this device.
